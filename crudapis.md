# REST API Development (CRUD APIs)

## Overview

REST APIs enable communication between the React frontend and Spring Boot backend by performing Create, Read, Update, and Delete (CRUD) operations.

---

## CRUD Operations

### Create

* Register User
* Upload Medical Report

### Read

* View Reports
* View Analysis Results

### Update

* Update User Profile
* Modify Report Details

### Delete

* Delete Medical Reports

---

## API Endpoints

| Method | Endpoint      | Description    |
| ------ | ------------- | -------------- |
| POST   | /register     | Register User  |
| POST   | /login        | User Login     |
| POST   | /reports      | Upload Report  |
| GET    | /reports      | View Reports   |
| PUT    | /profile      | Update Profile |
| DELETE | /reports/{id} | Delete Report  |

---

## Benefits

* RESTful Architecture
* JSON Data Exchange
* Easy Frontend Integration
* Scalable APIs
