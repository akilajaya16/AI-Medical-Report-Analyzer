# Entity & Repository Creation

## Overview

The Entity and Repository layers are responsible for mapping Java objects to database tables and performing database operations using Spring Data JPA.

---

## Objective

* Map database tables into Java Entity classes.
* Simplify database operations using JPA repositories.
* Reduce SQL code through Object Relational Mapping (ORM).

---

## Entity Classes

* User Entity
* MedicalReport Entity
* AnalysisResult Entity
* Parameter Entity
* ResultValue Entity

---

## Repository Interfaces

* UserRepository
* MedicalReportRepository
* AnalysisResultRepository
* ParameterRepository
* ResultValueRepository

---

## Benefits

* Automatic CRUD Operations
* Easy Database Management
* Improved Code Reusability
* Better Maintainability

---

## Expected Outcome

Entity classes accurately represent database tables, while repositories efficiently manage all database operations.
