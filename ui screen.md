# UI Screens

## 1. Login Screen

**Description:**
The Login Screen allows users to securely access the AI-Based Medical Report Analyzer system using their registered email and password.

**Features:**

* User Authentication
* Secure Login
* Forgot Password Option

---

## 2. Dashboard Screen

**Description:**
The Dashboard provides an overview of uploaded reports, analysis history, and system statistics.

**Features:**

* Recent Reports
* Analysis Summary
* Quick Navigation
* User Profile Access

---

## 3. Medical Report Upload Screen

**Description:**
Users can upload medical reports in PDF or image format for AI-based analysis.

**Features:**

* File Upload
* Report Type Selection
* Upload Confirmation

---

## 4. Report Analysis Screen

**Description:**
Displays extracted text and medical parameters identified from the uploaded report.

**Features:**

* OCR Extracted Data
* Parameter Detection
* Medical Information Display

---

## 5. Analysis Result Screen

**Description:**
Shows the AI-generated analysis results and highlights abnormal values.

**Features:**

* Normal/Abnormal Indicators
* Risk Level Detection
* Detailed Analysis Results

---

## 6. Health Summary Screen

**Description:**
Provides a simplified summary of the patient's medical report in an easy-to-understand format.

**Features:**

* Health Insights
* Key Findings
* Recommendations

---

## 7. Report History Screen

**Description:**
Stores all previously analyzed reports for future reference.

**Features:**

* View Past Reports
* Search Reports
* Download Analysis Results

---

## 8. User Profile Screen

**Description:**
Allows users to manage personal information and account settings.

**Features:**

* Profile Update
* Password Change
* Account Management

---

## 9. Admin Dashboard Screen

**Description:**
Enables administrators to monitor users, reports, and system performance.

**Features:**

* User Management
* Report Monitoring
* Activity Logs
* System Statistics

---
