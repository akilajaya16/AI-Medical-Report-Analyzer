# Frontend & Backend Integration

## Overview

The React frontend is integrated with the Spring Boot backend using REST APIs. Axios is used for sending HTTP requests and receiving JSON responses.

---

## Integration Flow

1. User performs an action in the React application.
2. React sends an API request to the backend.
3. Spring Boot processes the request.
4. Database operations are performed.
5. Backend returns a JSON response.
6. React updates the user interface.

---

## Integrated Modules

* User Registration
* User Login
* Medical Report Upload
* AI Analysis Results
* Health Summary
* Report History
