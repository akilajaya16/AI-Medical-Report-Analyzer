# Page Layouts

## 1. Login Page

**Purpose:** Allows users to securely access the system.

**Components:**

* Email Field
* Password Field
* Login Button
* Forgot Password Link

---

## 2. Dashboard Page

**Purpose:** Displays an overview of uploaded reports and analysis history.

**Components:**

* User Profile
* Recent Reports
* Analysis Statistics
* Navigation Menu

---

## 3. Medical Report Upload Page

**Purpose:** Enables users to upload medical reports in PDF or image format.

**Components:**

* File Upload Section
* Report Type Selection
* Submit Button

---

## 4. Report Analysis Page

**Purpose:** Displays extracted medical information and analysis results.

**Components:**

* Extracted Report Text
* Detected Parameters
* Abnormal Value Indicators
* Risk Assessment

---

## 5. Health Summary Page

**Purpose:** Provides a simplified explanation of the medical report.

**Components:**

* Health Summary
* Key Findings
* Recommendations
* Download Report Option

---

## 6. Report History Page

**Purpose:** Stores previously analyzed reports for future reference.

**Components:**

* Report List
* Upload Date
* Analysis Status
* View Details Button

---

## 7. User Profile Page

**Purpose:** Manages user information and account settings.

**Components:**

* Personal Information
* Password Management
* Profile Update Option

---

## 8. Admin Dashboard Page

**Purpose:** Allows administrators to monitor system activities.

**Components:**

* User Management
* Report Monitoring
* System Statistics
* Activity Logs

---
